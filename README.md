# Echo — Smiles in Mind

Landing page for **Echo**, an iOS app that helps you save your happiest moments and relive them whenever you want.

## About Echo

Echo is a companion app designed to capture the little moments that make you smile. Write a memory in seconds, share it in private groups with your closest circles, and watch your collection of moments grow into a living story.

## Project Structure

```
echo-web/
├── index.html        # Landing page
├── privacy.html      # Privacy policy
├── terms.html        # Terms of use
├── css/
│   └── style.css     # Styles
└── images/           # App screenshots & icons
```

## Run Locally

Open `index.html` in your browser — no build step required.
